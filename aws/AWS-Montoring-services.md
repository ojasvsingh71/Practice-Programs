# AWS Monitoring service

**AWS Cloudwatch**

It can monitor all services like Saas, Paas and Iaas.

Tracking, Monitoring and observability service like S3 requests, EC2 CPU usage, Lambda invocations

Monitors performances in form of Metrics and logs

Alarms and Dashboards monitoring

EC2, Lambda, RDS Monitoring

**SNS(Simple notification service)**

Eg:

Set an alarm if CPU consumption usage is >80%  then take an action and send a notification through SNS.

**Lambda(Used for Event triggering)**

Eg: Event triggers as msg when an amount is debited from account the msg is convert into voice.

**Polly**- converts text to voice.

**AWS CloudTrail**

API Activity logs

User Actions Tracking

Audit and Security

**AWS X-Ray**

Debug and Analyze

Dependency Map

Distributed Tracing
